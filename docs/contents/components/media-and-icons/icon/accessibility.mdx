## ARIA Roles and Attributes

| Component | Roles and Attributes | Usage                                                          |
| --------- | -------------------- | -------------------------------------------------------------- |
| `Icon`    | `aria-hidden="true"` | Excludes the element from the accessibility tree.              |
|           | `tabIndex`           | Set it to `0` to enable keyboard focus, or `-1` to disable it. |
