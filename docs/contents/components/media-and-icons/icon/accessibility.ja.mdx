## ARIAロールと属性

| コンポーネント | ロールと属性         | 使い方                                                                            |
| -------------- | -------------------- | --------------------------------------------------------------------------------- |
| `Icon`         | `aria-hidden="true"` | 要素をアクセシビリティツリーから除外します。                                      |
|                | `tabIndex`           | キーボードフォーカスを有効にしたい場合は`0`、無効にしたい場合は`-1`を設定します。 |
