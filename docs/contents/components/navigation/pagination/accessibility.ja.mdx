## ARIAロールと属性

| コンポーネント                   | ロールと属性        | 使い方                                                                                                                                     |
| -------------------------------- | ------------------- | ------------------------------------------------------------------------------------------------------------------------------------------ |
| `Pagination`                     | `role="navigation"` | ページ間のナビゲーション領域であることを示します。                                                                                         |
| `PaginationItem` <Tag>内部</Tag> | `aria-label`        | それぞれに`"Go to first page"`, `"Go to previous page"`, `"Go to next page"`, `"Go to last page"`と`"Go to page <em>N</em>"`を設定します。 |
