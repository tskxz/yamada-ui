`Select`は、アクセシビリティに関して[WAI-ARIA - Combobox Pattern](https://www.w3.org/WAI/ARIA/apg/patterns/combobox/)に従います。

## キーボード操作

| キー             | 説明                                                                                                       | 状態 |
| ---------------- | ---------------------------------------------------------------------------------------------------------- | ---- |
| `Tab`            | フォーカスがセレクトボックスに移動すると、最初の項目をフォーカスします。                                   | -    |
| `ArrowUp`        | 無効ではない前のオプションをフォーカスします。最初の項目の場合は無効ではない最後の項目をフォーカスします。 | -    |
| `ArrowDown`      | 無効ではない次のオプションをフォーカスします。最後の項目の場合は無効ではない最初の項目をフォーカスします。 | -    |
| `Space`, `Enter` | フォーカスしているオプションをアクティブにします。                                                         | -    |
| `Home`           | 無効ではない最初のオプションをフォーカスしてアクティブにします。                                           | -    |
| `End`            | 無効ではない最後のオプションをフォーカスしてアクティブにします。                                           | -    |

## ARIAロールと属性

| コンポーネント                | ロールと属性                    | 使い方                                                                                                    |
| ----------------------------- | ------------------------------- | --------------------------------------------------------------------------------------------------------- |
| `Select`                      | `role="combobox"`               | コンボボックスであることを示します。                                                                      |
|                               | `aria-label="Select an option"` | `aria-label`が設定されていない場合は、`"Select an option"`が設定されます。                                |
|                               | `aria-activedescendant`         | リストボックスのオプションにフォーカスがある場合、現在選択されているオプションの`id`を設定します。        |
|                               | `aria-controls`                 | 関連したリストボックスの`id`を設定します。                                                                |
|                               | `aria-expanded`                 | オプションのパネルが展開されている場合は`"true"`を設定し、折りたたまれている場合は`"false"`を設定します。 |
| `SelectList` <Tag>内部</Tag>  | `role="listbox"`                | リストボックスであることを示します。                                                                      |
| `Option` <Tag>内部</Tag>      | `role="option"`                 | オプションであることを示します。                                                                          |
|                               | `aria-checked`                  | オプションが選択されている場合は`"true"`を設定し、選択されていない場合は`"false"`を設定します。           |
|                               | `aria-disabled`                 | `isDisabled`が設定されている場合は`"true"`を設定します。                                                  |
| `OptionGroup` <Tag>内部</Tag> | `role="group"`                  | グループであることを示します。                                                                            |
|                               | `aria-label`                    | グループで表示しているラベルを設定します。                                                                |
