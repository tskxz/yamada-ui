The `ColorPicker` follows the [WAI-ARIA - Combobox Pattern](https://www.w3.org/WAI/ARIA/apg/patterns/combobox/) for accessibility.

Setting an `aria-label` on the `ColorPicker` allows it to be read by screen readers.

```tsx live=false
<ColorPicker aria-label="Fill color" defaultValue="#4387f4ff" />
```

## Keyboard Navigation

| Key                    | Description                              | State |
| ---------------------- | ---------------------------------------- | ----- |
| `Tab`, `Shift` + `Tab` | Focus on the element and open the popup. | -     |

## ARIA Roles and Attributes

| Component     | Role and Attribute | Usage                                           |
| ------------- | ------------------ | ----------------------------------------------- |
| `ColorPicker` | `aria-readonly`    | Set to `"true"` when `isReadOnly` is specified. |
|               | `aria-disabled`    | Set to `"true"` when `isDisabled` is specified. |
|               | `aria-invalid`     | Set to `"true"` when `isInvalid` is specified.  |
