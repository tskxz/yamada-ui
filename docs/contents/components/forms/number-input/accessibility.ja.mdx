## キーボード操作

| キー        | 説明                                                | 状態 |
| ----------- | --------------------------------------------------- | ---- |
| `ArrowUp`   | 値を増加させます。増加する値は`step`で指定します。  | -    |
| `ArrowDown` | 値を減少させます。減少する値は`step`で指定します。  | -    |
| `Home`      | 値を最小値に設定します。最小値は`min`で指定します。 | -    |
| `End`       | 値を最大値に設定します。最大値は`max`で指定します。 | -    |

## ARIAロールと属性

| コンポーネント | ロールと属性        | 使い方                                                   |
| -------------- | ------------------- | -------------------------------------------------------- |
| `NumberInput`  | `role="spinbutton"` | スピンボタンであることを示します。                       |
|                | `aria-valuemin`     | コンポーネントが取りうる値の最小値を支援技術に伝えます。 |
|                | `aria-valuemax`     | コンポーネントが取りうる値の最大値を支援技術に伝えます。 |
|                | `aria-valuenow`     | コンポーネントの現在の値を支援技術に伝えます。           |
|                | `aria-valuetext`    | 人間が読み取り可能な値を支援技術に伝えます。             |
