## キーボード操作

| キー             | 説明                                                                                              | 状態 |
| ---------------- | ------------------------------------------------------------------------------------------------- | ---- |
| `Tab`            | フォーカスがカレンダーに移動すると、ヘッダーをフォーカスします。                                  | -    |
| `ArrowLeft`      | フォーカスを左の日付に1つ移動します。行の一番左にフォーカスがある場合、フォーカスは移動しません。 | -    |
| `ArrowRight`     | フォーカスを右の日付に1つ移動します。行の一番右にフォーカスがある場合、フォーカスは移動しません。 | -    |
| `ArrowUp`        | フォーカスを上の日付に1つ移動します。列の一番上にフォーカスがある場合、フォーカスは移動しません。 | -    |
| `ArrowDown`      | フォーカスを下の日付に1つ移動します。列の一番下にフォーカスがある場合、フォーカスは移動しません。 | -    |
| `Space`, `Enter` | フォーカスしている日付をアクティブにします。                                                      | -    |
| `Home`           | 月の最初の日付をフォーカスします。                                                                | -    |
| `End`            | 月の最後の日付をフォーカスします。                                                                | -    |

## ARIAロールと属性

| コンポーネント              | ロールと属性  | 使い方                                                  |
| --------------------------- | ------------- | ------------------------------------------------------- |
| `Month` <Tag>内部</Tag>     | `role="grid"` | グリッドであることを示します。                          |
|                             | `aria-label`  | 選択中の年月を設定します。(例: `October 2024`)          |
| `MonthList` <Tag>内部</Tag> | `role="grid"` | グリッドであることを示します。                          |
|                             | `aria-label`  | 選択中の年を設定します。(例: `2024`)                    |
| `YearList` <Tag>内部</Tag>  | `role="grid"` | グリッドであることを示します。                          |
|                             | `aria-label`  | 表示中の年の範囲を設定します。(例: `From 2021 to 2032`) |
