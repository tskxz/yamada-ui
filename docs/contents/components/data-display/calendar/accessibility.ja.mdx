## キーボード操作

| キー             | 説明                                                                                                                                                                               | 状態 |
| ---------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ---- |
| `Tab`            | フォーカスがカレンダーに移動すると、ヘッダーをフォーカスします。もう一度押下すると、選択された日付にフォーカスします。選択されていない場合はその月の最初の日付にフォーカスします。 | -    |
| `ArrowUp`        | フォーカスを前のセルに1つ移動します。                                                                                                                                              | -    |
| `ArrowDown`      | フォーカスを次のセルに1つ移動します。                                                                                                                                              | -    |
| `Space`, `Enter` | フォーカスしているセルをアクティブにします。                                                                                                                                       | -    |

## ARIAロールと属性

| コンポーネント              | ロールと属性  | 使い方                                                  |
| --------------------------- | ------------- | ------------------------------------------------------- |
| `Month` <Tag>内部</Tag>     | `role="grid"` | グリッドであることを示します。                          |
|                             | `aria-label`  | 選択中の年月を設定します。(例: `October 2024`)          |
| `MonthList` <Tag>内部</Tag> | `role="grid"` | グリッドであることを示します。                          |
|                             | `aria-label`  | 選択中の年を設定します。(例: `2024`)                    |
| `YearList` <Tag>内部</Tag>  | `role="grid"` | グリッドであることを示します。                          |
|                             | `aria-label`  | 表示中の年の範囲を設定します。(例: `From 2021 to 2032`) |
