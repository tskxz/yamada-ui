---
title: useBreakpointState
description: >-
  `useBreakpointState` is a custom hook that takes a responsive object as an
  initial state and returns a state corresponding to the current breakpoint.
package_name: "@yamada-ui/use-breakpoint"
with_description: true
---

## Import

```ts
import { useBreakpointState } from "@yamada-ui/react"
```

## Usage

```tsx functional=true
const [value, setValue] = useBreakpointState({
  base: 1,
  md: 2,
  sm: 3,
})

return (
  <Button onClick={() => setValue((prev) => prev + 1)}>
    The current value is "{value}"
  </Button>
)
```

:::note status=warning
The keys of the object define the keys set in the theme's [breakpoint](/styled-system/theming/default-theme#breakpoint).
:::

:::note
If you want to learn more about responsive styles, please check [here](/styled-system/responsive-styles).
:::
